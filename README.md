# Seuils physiologiques adaptés chez les patients âgés

## Présentation

Ce projet vise à analyser l’adéquation des seuils physiologiques standards
chez les patients âgés (≥ 65 ans) à partir de la base de données clinique **MIMIC-IV**.

L’objectif est de structurer une analyse exploratoire menant à la définition
potentielle de seuils plus adaptés à la physiologie des personnes âgées.

## Données

Les données utilisées proviennent de la base **MIMIC-IV**.
Aucune donnée brute ni information permettant d’identifier un patient
n’est incluse dans ce dépôt.

## État d’avancement

* **Phase 1** : Bases en R et visualisation
* **Phase 2** : Exploration démographique des patients âgés de ≥ 65 ans
* **Phase 3** : Analyse des signes vitaux en soins intensifs (ICU) — *en cours*

## Structure

* `R/` : scripts d’analyse
* `data/` : données brutes et transformées (non versionnées)
